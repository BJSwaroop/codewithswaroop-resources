5-STEP GUIDE
Why a portfolio? 80% of freshers don't have one. Having even a basic portfolio puts you ahead of the majority. Recruiters remember visual proof over resume bullet points.
1

Create a GitHub Repository

Go to github.com → New Repository. Name it yourusername.github.io (exactly your GitHub username). This special name tells GitHub to host it as a website automatically.

Takes 1 minute
2

Add Your index.html

Create an index.html file in the repo. This is your homepage. Start with a simple template:

  • Hero section — Name, tagline, profile photo
  • About — 2-3 sentences about you
  • Projects — Top 3 projects with links
  • Skills — Technical skills list
  • Contact — Email + LinkedIn + GitHub links
Core structure
3

Style It with CSS

Keep it clean and minimal. Dark theme or white — pick one. You don't need a framework:

  • Use Google Fonts (Inter or Poppins look modern)
  • Responsive design — test on mobile
  • No animations needed — clean beats flashy
  • Use CSS Grid or Flexbox for layout
Keep it simple
4

Enable GitHub Pages

Go to repo Settings → Pages → Source: "Deploy from a branch" → Branch: main → Save. Your site goes live at https://yourusername.github.io within 2 minutes.

Free hosting forever
5

Add It Everywhere

Now that you have a live URL, put it in:

  • Resume header — right after your email
  • LinkedIn — Featured section + Contact info
  • GitHub profile README
  • Email signature
  • Every job application
Maximum visibility

Free Templates to Start With

  • HTML5 UP — html5up.net (free, responsive)
  • GitHub Pages Themes — built-in Jekyll themes
  • Astro Portfolio Template — modern, fast, free
  • Or just build from scratch — recruiters appreciate that more
